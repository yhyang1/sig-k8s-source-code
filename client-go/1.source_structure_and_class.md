## kubernetes的client-go目录结构

### 1.14.0 版本目录结构如下
```
client-go
├── Godeps
├── deprecated-dynamic
├── discovery
├── dynamic
├── examples
├── informers
├── kubernetes
├── kubernetes_test
├── listers
├── pkg
├── plugin
├── rest
├── restmapper
├── scale

├── third_party

├── transport

```

### 目录结构分类
* discovery, dynamic, rest, scale, kubernetes 为各种客户端
* informers
* listers
* plugin
* tools
* transport
* util
